You are developing an HTML page that includes the following code. <h1 id="header">A Static Page</h1> You need to modify the content of the H1 element dynamically by using JavaScript. Which code segment should you use? Example Change the HTML content of a <h1> </h1> element with id="header": document.getElementById("header").innerHTML = "A dynamic page";
